## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

```
# run the application using Docker by running the command below:
docker-compose up --build Or docker-compose up
```

## Swagger

Application is running on port 3000 and swagger accessible on http://localhost:3000/swagger
